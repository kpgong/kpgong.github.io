最长环路 - 算法典中典3/90

2021年06月11日 3 min read # 算法典中典 # 程序设计竞赛

题目来源：AtCoder-最长环路

难度：★★★★☆☆☆

问题描述

有一无向图，带有 NNN 个顶点，(N−1)(N-1)(N−1) 条边。任意两个顶点之前存在连通的路径。现选取两个不同的顶点，添加新的一条边连通它们，求 SSS 的最大值，SSS 为“在不经过重复边的情况下，从一个顶点到它本身所经过的边的数目”。顶点和边均用正整数编号。

数据范围

  • 3≤N≤100003\le N \le 100003≤N≤10000

数据格式

输入:

N
A_1 B_1
A_2 B_2
...

输入的 Ai    BiA_i\;\;B_iAi​Bi​ 表示顶点 Ai    BiA_i\;\;B_iAi​Bi​ 间存在一条边。

输出:

SSS 的最大值

测试样例

输入:

3
1 2
2 3

输出:

3

解析

边的数目比顶点数目少一，且任意两条边之前存在连通路径。所以这是一棵树（想一想为什么）。

树无环。依据题意，待添加的边一定会构造出一个唯一的环。所以，如果在顶点 AAA 和 BBB 添加新的边，那么 SSS 的值就等于原树中 AAA 和 BBB 的路径长度加 111。

原问题转化为求树中的最长路径。树中的最长路径称为直径。树的直径的求解方法如下：

  1. 任选一个顶点 AAA，求出诸顶点到 AAA 的路径长；
  2. 选取一个顶点 BBB，BBB 是距离 AAA 最远的顶点（之一）；
  3. 求出诸顶点到 BBB 的路径长；
  4. 选取一个顶点 CCC，CCC 是距离 BBB 最远的顶点（之一），BBB 和 CCC 之间的距离就是树的直径。

要求出诸顶点到指定顶点的路径长，可以考虑用 DFS 或 BFS，时间复杂度为 O(N)\Omicron\left(N\right)O(N)。

代码

struct Solution
{
#define tfor(i, n) for (int i = 0; i < (n); ++i)
    int n;
    vector<vector<int> > mat;
    vector<int> len;
    vector<bool> vis;

    void dfs(int x)
    {
        int ret = 0;
        vis[x] = true;
        for (auto& i : mat[x])
        {
            if (!vis[i])
            {
                len[i] = len[x] + 1;
                dfs(i);
            }
        }
    }

    void input()
    {
        int a, b;
        cin >> n;
        mat = vector<vector<int> >(n);
        tfor(i, n - 1)
        {
            cin >> a >> b;
            --a, --b;
            mat[a].push_back(b);
            mat[b].push_back(a);
        }
    }

    Solution()
    {
        input();
        len = vector<int>(n);
        vis = vector<bool>(n);
        dfs(0);
        int f = 0;
        tfor(i, n)
        {
            if (len[i] > len[f]) f = i;
        }
        len = vector<int>(n);
        vis = vector<bool>(n);
        dfs(f);
        tfor(i, n)
        {
            if (len[i] > len[f]) f = i;
        }
        cout << len[f] + 1 << endl;
    }
#undef tfor
};

int main()
{
    Solution();
    return 0;
}
