括号大全 - 算法典中典2/90

2021年06月10日 2 min read # 算法典中典 # 程序设计竞赛

题目来源：AtCoder-括号大全

问题描述

按字典序输出所有长度为 NNN 的合法括号串。

难度：★★★☆☆☆☆

数据范围

1≤N≤201\le N \le 201≤N≤20

测试样例

stdin_1:

4

stdout_1:

(())
()()

stdin_2:

3

stdout_2:

可以什么都不输出

解析1

1≤N≤201\le N \le 201≤N≤20

范围足够小->枚举（最多 104,8576104,8576104,8576 个串）然后筛选、排序。

解析2

合法括号串的前缀中的左括号数不少于右括号数->剪枝搜索

代码

res = []
n = int(input())

# l和r分别为串s中的左括号数和右括号数
def gen(s, l, r):
    global res, n
    if (r == n):
        res.append(s)
    else:
        # 先考虑添加左括号，再考虑添加右括号
        # 保证了按字典序生成，不必再排序
        if (l < n): # 左括号数不能超过n
            gen(s + "(", l + 1, r)
        if (r < l): # 前缀中的左括号数不少于右括号数
            gen(s + ")", l, r + 1)

if (n % 2 == 0):
    n = n // 2
    gen("", 0, 0)
    for x in res:
        print(x)
